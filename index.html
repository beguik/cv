En esta sopa de letras hay ocultas 11 palabras relacionadas con el mundo del desarrollo web, cuando seleccione una de ellas aparecerá en la derecha la experiencia que tengo con el lenguaje o herramienta seleccionado. Si se da por vencido puede darle a resolver y se mostrarán todas.
CSS: El diseño de una web está ligado a este lenguaje que define sus estilos, por tanto he trabajado este lenguaje de manera transversal en cada página que he diseñado. He trabajado también con framework como Bootstrap o Foundation.
HTML: Ni que decir que HTML es un lenguaje básico para el desarrollo web y que por tanto lo he ido trabajando en cada página que he realizado, incluyendo los nuevos estandares de HTML5.
JavaScript: Esta misma página está construida con elementos JS que la hacen interactiva. Personalmente me gusta mucho este lenguaje que permite realizar proyectos muy atractivos visualmente y que facilitan la interacción con el usuario. Además de los elementos que trabajo en esta web con javaScript he trabajado también la validación de campos en el cliente y peticiones Ajax al servidor.
JQuery: En algunas ocasiones el trabajo con Jquery facilita la interación con el DOM y por eso en algunos proyectos he preferido trabajar algunas partes con esta tecnología.
Java: Mi primera aproximación al mundo de la programación fue con este lenguaje de programación y he de decir que no fue un comienzo fácil, pero una vez interiorizados los conceptos básicos la curva de aprendizaje creció exponencialmente. He trabajado con java la creación de aplicaciones, de sockets, conexión a base de datos y el diseño de interfaz gráfica con Java Swing.
PHP: Con php he trabajado creando aplicaciones del lado del servido, la validacion de objetos y el acceso a base de datos. He trabajado con php puro y también aprendí a manejarme en el framework de Laravel.
Python:Este lenguje de programación lo aprendí de forma totalmente autodidacta, cuando supe que en la empresa, Tangrambpm, donde iba a realizar las prácticas del grado trabajaba exclusivamente con este lenguaje que no se impartía en el ciclo decidí realizar el proyecto fin de ciclo, consistente en un portal inmobiliario, en python para ir adquiriendo soltura. Posteriormente, las prácticas y el posterior trabajo realizado en esta empresa ha sido desarrollado integramente en python con el framework de Django.
GIT: El control de versiones es una herramienta fundamental para el desarrollo en equipo, en el último trabajo realizado trabajamos tanto con Mercurial como con Git
SQL: Las consultas a base de datos las he trabajado con mySQL y postgreSQL.
Wordpress: Antes de dedicarme al desarrollo Web he trabajado como autónoma y he desarrollado varias web en wordpress con Elementor y Woocomerce. También en casos puntuales he trabajado con Prestashop en el marco de un marketplace.
Redes Sociales: He trabajado también las redes sociales y la imagen de la empresa Clau Complementos, atendiendo a las necesiades de la misma.
